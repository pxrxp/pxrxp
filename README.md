### `💡 A Catalog of Curiosities`

<details>
<summary>Ideas and domains that I find endlessly fascinating.<br>Click to expand</summary>
<br>

| Domain                     | The Core Joy                        | My Perspective                                                                                                                                                                                               |
| -------------------------- | ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **`⚙️ Systems Programming`**  | Building on solid ground.           | I like the confidence that comes from tools like Rust. The compiler helps you build high-performance code without the constant fear of memory bugs. Who has time for segfaults?                              |
| **`λ Functional Logic`**      | A moment of pure clarity.           | I appreciate the paradigm shift that languages like Haskell offer. It's a move away from describing steps and towards defining pure data transformations. The code itself becomes a beautiful and verifiable proof of its own logic. |
| **`🎨 Low-Level Graphics`**    | Creating something from nothing.    | It's magical to turn abstract math and logic into tangible visuals on a screen. Starting with a blank canvas and building a world pixel by pixel feels fundamental.                                        |
| **`🧠 Machine Learning`**     | Demystifying the black box.         | I love digging into the models to understand *why* they work. It's fascinating to see the underlying math come to life and watch a system actually learn from data.                                        |
| **`🔧 Compilers & Interpreters`** | Peeking behind the curtain.       | Understanding how our code is translated into something a machine can execute is incredibly empowering. It removes a layer of magic and deepens your respect for the craft.                             |
| **`🖥️ Operating Systems`**    | Knowing what's happening underneath. | It's satisfying to understand how the computer manages memory, schedules processes, and handles I/O. It’s the foundation that all other software is built upon.                                        |
| **`🌍 Full-Stack Architecture`** | Making an idea real.                | This is the final step where all the abstract systems come together. It's the satisfaction of building a complete, tangible product that people can actually use.                                      |

</details>
